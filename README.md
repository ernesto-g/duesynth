# MINIBRUTE Clone with Arduino DUE

![alt text](https://raw.githubusercontent.com/ernesto-g/duesynth/master/Pictures/6.jpg)


Supported features:
  - Pitch Wheel
  - Mod Wheel
  - Octave selection
  - Vibrato
  - Sequencer
  - Waveforms:
    - Sub osc
    - Square
    - Saw
    - Triangular
    - Noise	
  - Waveform mods:
    - Ultrasaw
    - PWM
    - Metalizer
  - Analog VCF:
    - HPF / BPF / LPF
    - Cutoff [knob or modwheel]
    - Resonance
    - Envelope
  - LFO	
  - Inputs:
    - Line
    - MIDI	
  - Outputs:
    - Line
    - Headphones
    - MIDI
    - USB MIDI
	
Still not implemented features:  
  - Vibrato
  - Sequencer
  - Configuration screen
  - Sequencer screen
  - MIDI In
  - MIDI Out

  
Not Supported features:
  - Brute factor
  - Fine tune
  - Mod wheel LFO Amt
  - Bend range
  - Glide
  - Aftertouch
  - Arpeggiator (was changed by microbrute sequencer)
